## üîë Features
- **Cross-Platform App**
  - Built with **Flutter**, runs on both Android and iOS.  
- **Backend & Database**
  - Integrated with **Firebase Authentication** for secure login/signup.  
  - **Cloud Firestore** for real-time data storage and syncing across users.  
- **User Experience**
  - Intuitive and responsive UI designed with Material Design principles.  
  - Real-time collaboration features (e.g., shared food boards, group updates).

## üöÄ Getting Started

### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install)  
- [Firebase Project](https://firebase.google.com/) linked to your app  
- Android Studio / VS Code with Flutter extension  

### Setup

#### 1. Clone the repository:
git clone https://github.com/YourUsername/FoodBNB.git
cd FoodBNB

#### 2. Install dependencies:
flutter pub get

#### 3. Configure Firebase:
    - Add google-services.json (Android) in android/app/
    - Add GoogleService-Info.plist (iOS) in ios/Runner/

#### 4. Run the app:
flutter run
